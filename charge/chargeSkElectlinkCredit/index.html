sk일렉링크 충전 크레딧 구매 및 판매 방법

sk일렉링크

카드 할인 한도가 남는데 충전을 많이 하지 못했다면 매달 할인이 리셋되어 뭔가 아까운 느낌이 있었습니다.
sk일렉링크는 크레딧 충전 시 5% 할인 혜택을 제공합니다.
고속도로 휴게소에 설치된 급속 충전기가 많아 미리 크레딧을 구매해두시면 장거리 여행 시 아주 유용합니다.
집밥이 sk일렉링크 충전기라면 카드 할인 계산하며 쓰는 것 보다 매달 초에 크레딧 충전하여 사용하는 것도 좋은 방법입니다.
(반대로 장터에서 구매하면 조금이라도 저렴하게 사용 가능한거겠죠?)

크레딧 구매 및 판매 방법

1. sk일렉링크 앱에서 스토어로 이동

2. 충전 크레딧 구매하기로 이동

3. 구매금액에서 금액 부분을 왼쪽으로 슬라이드

4. 직접입력 선택

5-1. (충전시) 구매금액 직접 입력하고 나에게 보내기 체크 후 구매시 유의사항 동의 선택 후 결제

5-2.(판매시) 연락처 가져오기에서 구매자 연락처 선택 후 구매시 유의사항 동의 선택 후 결제

5-2-1.(판매완료) 크레딧 구매가 완료되었습니다.

sk일렉링크 친구 초대

위 게시글이 도움이 되셨다면 일렉링크 가입 시 아래 레퍼럴 코드를 입력하시면 감사하겠습니다.
(친구 초대 시 1,000원 크레딧 제공)
여기를 클릭하셔서 가입하시거나,

초대코드 입력 방법

1. SK일렉링크 앱을 설치하신 후 스토어 선택 후 친구 초대하기 선택

2. 코드입력 선택

3. 초대코드 입력 [초대 코드 : KS1BXP]

초대코드 입력 시 1,000 크레딧을 받을 수 있습니다. [초대 코드 : KS1BXP]
