# 테슬라 소유권 이전 완벽 가이드: 앱으로 5분 만에 계정 옮기기 (중고차 판매, 레퍼럴 활용)

테슬라 차량을 판매하거나 다른 사람에게 양도할 때, 가장 중요한 절차 중 하나는 바로 **차량 계정 소유권 이전**입니다. 다행히 이 모든 과정은 테슬라 공식 앱을 통해 누구나 간편하게 진행할 수 있습니다.

최근에는 테슬라 레퍼럴(Referral) 프로그램을 활용하기 위해 새 계정으로 차량을 구매한 뒤, 기존에 사용하던 메인 계정으로 차량을 옮기는 분들도 많습니다. 이런 경우에도 소유권 이전은 필수입니다.

이 글에서는 테슬라 앱을 통해 차량 소유권을 이전하는 전체 과정을 스크린샷과 함께 단계별로 쉽고 명확하게 알려드립니다.

## 테슬라 소유권 이전 절차: Step-by-Step

소유권 이전 과정은 크게 **① 메뉴 진입 → ② 이전 시작 → ③ 양수자 정보 입력 및 인증 → ④ 완료 확인** 4단계로 나눌 수 있습니다.

### 1단계: 앱 메뉴에서 '소유권 이전' 항목 찾기

![테슬라 앱 메인 화면에서 좌측 상단 메뉴 선택](/images/teslaAccountTransfer/teslaAccountTransfer-1.PNG "테슬라 앱 메뉴")
먼저 테슬라 앱을 실행하고, 좌측 상단의 메뉴(≡) 아이콘을 탭합니다.

![앱 메뉴에서 '내 제품' 선택](/images/teslaAccountTransfer/teslaAccountTransfer-2.PNG "'내 제품' 메뉴")
나타난 메뉴에서 **내 제품**을 선택합니다.

![소유권을 이전할 차량 선택](/images/teslaAccountTransfer/teslaAccountTransfer-3.PNG "이전할 차량 선택")
'내 제품' 목록에서 소유권을 이전하려는 차량을 선택하세요.

### 2단계: 소유권 이전 절차 시작하기

![차량 상세 화면에서 '소유권 삭제 및 이전' 선택](/images/teslaAccountTransfer/teslaAccountTransfer-4.PNG "'소유권 삭제 및 이전' 메뉴")
차량 상세 화면에서 **'소유권 삭제 및 이전'** 옵션을 찾아 누릅니다.

![소유권 이전 안내 화면에서 '시작하기' 버튼 클릭](/images/teslaAccountTransfer/teslaAccountTransfer-5.PNG "'시작하기' 버튼")
안내 화면이 나타나면 **'시작하기'** 버튼을 눌러 절차를 시작합니다.

!['이 차량을 판매하셨나요?'에서 '예' 선택, '개인'에게 판매 선택](/images/teslaAccountTransfer/teslaAccountTransfer-6.PNG "소유권 이전 목적 선택")
소유권 이전 목적을 묻는 질문입니다.

- `이 차량을 판매하셨나요?` → **`예`**
- `어느 업체에게 판매했나요?` → **`개인`** (개인 간 거래 기준)

### 3단계: 양수자(인수자) 정보 입력 및 인증

![차량을 받을 사람의 이름과 이메일 주소 입력](/images/teslaAccountTransfer/teslaAccountTransfer-7.PNG "양수자 정보 입력")
차량을 받을 사람(양수자)의 **이름**과 **테슬라 계정 이메일 주소**를 정확하게 입력합니다.

![계정 보안을 위한 이메일 인증 코드 입력](/images/teslaAccountTransfer/teslaAccountTransfer-8.PNG "이메일 코드 인증")
보안을 위해 현재 계정(양도자)의 이메일로 전송된 **인증 코드를 입력**합니다.

![정보 확인 후 '확정' 버튼을 눌러 요청 제출](/images/teslaAccountTransfer/teslaAccountTransfer-9.PNG "소유권 이전 요청 제출")
코드 입력 후 **'확정'** 버튼을 누르면 소유권 이전 요청이 최종 제출됩니다.

### 4단계: 이전 완료 확인

![차량 삭제 요청이 접수되었다는 확인 이메일](/images/teslaAccountTransfer/teslaAccountTransfer-10.PNG "차량 삭제 확인 이메일")
요청이 제출되면 현재 계정으로 '차량 삭제 요청 접수' 확인 메일이 발송됩니다.

![앱에서 차량이 사라진 초기 화면](/images/teslaAccountTransfer/teslaAccountTransfer-11.PNG "앱에서 차량 삭제 확인")
잠시 후, 현재 계정의 앱에서 해당 차량이 사라진 것을 확인할 수 있습니다. 이로써 **양도자의 절차는 모두 완료**되었습니다.

![양수자 계정 앱에 새로 추가된 차량 확인](/images/teslaAccountTransfer/teslaAccountTransfer-1.jpeg "양수자 계정에서 차량 확인")
마지막으로, 차량을 **인수받은 사람의 계정**으로 앱에 로그인하면 위와 같이 차량이 성공적으로 추가된 것을 볼 수 있습니다.

## 마무리

테슬라 차량 소유권 이전은 생각보다 훨씬 간단합니다. 앱 하나로 모든 절차를 마칠 수 있어 중고차를 판매하거나 가족에게 양도할 때 매우 편리합니다.

특히, 레퍼럴 혜택을 위해 계정을 분리하여 차량을 관리하는 분들이라면 이 소유권 이전 방법을 정확히 알아두는 것이 중요합니다. 불필요한 불편함 없이 스마트한 테슬라 라이프를 즐기시길 바랍니다.
